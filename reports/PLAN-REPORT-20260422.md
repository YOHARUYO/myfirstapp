## 3. 주요 결정 사항

### 3-1. AI 태깅/요약 진행 표시 (#1)

**문제**: AI 태깅·요약 실행 중 사용자에게 진행 상황이 보이지 않음.

**결정**:
| 상황 | 표시 방식 |
|------|----------|
| 5단계 "AI 재태깅" 버튼 | 버튼 → 로딩 스피너 + "태깅 중…" 전환, 완료 시 토스트 |
| 5→6 전환 (요약 생성) | 전체 화면 로딩 오버레이 + "AI 요약 생성 중…" |

### 3-2. 키보드 명령어 변경 (#4)

**문제**: 블록 내 줄바꿈 방법이 없고, Shift+Enter=분할이 일반 텍스트 에디터 관례와 반대.

**결정**:
| 키 | 변경 전 | 변경 후 |
|---|---------|---------|
| Shift+Enter | 블록 분할 | **블록 내 줄바꿈** |
| Ctrl+Enter (Cmd+Enter) | 없음 | **블록 분할** |
| Backspace (첫 블록) | 400 에러 | **무시 (가드 추가)** |

### 3-3. 섹션 제목-콘텐츠 위계 강화 (#5)

**문제**: Display 제목과 콘텐츠의 경계가 시각적으로 모호.

**결정**: 서브텍스트(Small, text-secondary) + 콘텐츠 배경색 블록(bg-bg-subtle rounded-xl p-6) 조합. design-system.md에 패턴 + 페이지별 서브텍스트 예시 추가.

### 3-4. Slack 인사 문구 여러 줄 + 이모티콘 (#7)

**문제**: 단일 행 input으로 줄바꿈·이모티콘 불가.

**결정**: textarea(최소 3줄, 자동 확장) + OS 이모티콘 + `:emoji_name:` 문법 허용.

### 3-5. 히스토리 회의록 삭제 (#8)

**문제**: 완료된 회의록을 삭제하는 방법이 없음.

**결정**:
- 히스토리 상세 하단 `[🗑 삭제]` 버튼
- 확인 모달 (Slack 전송 이력 있으면 별도 안내)
- `DELETE /api/meetings/{id}` — meeting JSON + 오디오 + export md 삭제

### 3-6. 7단계 Slack 미리보기 실시간 반영 (#13 보완)

**문제**: 6단계 수정 내용이 7단계 미리보기에 반영되지 않음.

**결정**: 7단계 진입 시 서버에서 최신 데이터 fetch (캐시 금지). 6↔7 왕복 시 항상 재생성.
