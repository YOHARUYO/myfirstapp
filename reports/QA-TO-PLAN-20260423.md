## 1. 이미 개발에 전달된 변경 사항 — 기획 문서 동기화 필요 (10건)

아래 항목은 QA 검수에서 발견되어 개발에 수정 프롬프트로 전달된 것들이야.
코드는 변경되었지만 기획 문서에는 아직 반영 안 된 상태이므로 동기화 필요.

### 1-1. 마이크 민감도 범위 변경
- 기존: 0.5x ~ 3.0x
- 변경: 0.5x ~ 10.0x
- 적용: 3단계 녹음 회의 정보 패널 + 설정 화면
- 5x 이상 시 "오디오 왜곡 가능" 경고 표시
- 반영 위치: decisions.md 3단계 "마이크 민감도" + 설정 화면

### 1-2. 주소록 레이아웃 변경
- 기존: 1열 리스트 (한 줄에 이름 하나)
- 변경: 칩(태그) 형태의 flex-wrap 그리드
- 참여자와 장소 모두 동일 적용
- 반영 위치: decisions.md 설정 화면 "주소록", design-system.md (컴포넌트 패턴 추가 권장)

### 1-3. 텍스트 블록 편집 방식 변경
- 기존: textarea 고정 크기 + 내부 스크롤
- 변경: textarea 자동 확장 (overflow hidden, 스크롤 없음)
- 3단계 녹음 + 5단계 편집 공통
- 반영 위치: decisions.md "인라인 편집 UX" 섹션

### 1-4. 합치기 버튼 위치 변경
- 기존: 블록 내부에 인라인 (블록 크기 변화)
- 변경: 블록 우측 상단에 absolute 오버레이 (블록 크기 비영향)
- 반영 위치: decisions.md 5단계 "호버 ⋮ 메뉴"

### 1-5. 복구 배너 스타일 변경
- 기존: bg-bg-subtle (연회색) — 중요성 인지 어려움
- 변경: bg-warning-bg + border-warning/20 (경고 스타일, 📌 아이콘)
- 반영 위치: decisions.md 1단계 "복구 배너"

### 1-6. 침묵 타임아웃 변경
- 기존: 1.5초 (SILENCE_TIMEOUT_MS = 1500)
- 변경: 2.5초 (SILENCE_TIMEOUT_MS = 2500)
- 이유: 1.5초가 너무 짧아 생각하며 말할 때 문장이 중간에 끊김
- 반영 위치: technical-design.md 6절 Web Speech 관련 상수

### 1-7. 설정 저장 경로 → 7단계 반영
- 설정의 export_path 변경이 7단계 "로컬 저장"에 실시간 반영되도록 수정
- 7단계 진입 시 settings에서 경로 로드
- 반영 위치: decisions.md 7단계 "로컬 저장" + technical-design.md

### 1-8. Slack 채널 목록 새로고침
- 7단계 채널 드롭다운 옆에 새로고침(↻) 버튼 추가
- 채널 0개일 때 "/invite" 안내 문구
- 반영 위치: decisions.md 7단계 "Slack 채널 & 스레드 선택"

### 1-9. Slack 유저 ID 코드 표시 개선
- 스레드 메시지 목록에서 유저 코드(U1234) 대신 "사용자(마지막4자리)" 표시
- 반영 위치: decisions.md 7단계 "스레드 선택"

### 1-10. API 설정 표시 로직 변경

- 반영 위치: technical-design.md 4-9절 설정 API

아래 항목은 사용자가 요청했으나 기획 문서 신규 추가가 필요해서 개발 전달을 보류한 것.

### 2-1. 회의 템플릿 순서 드래그

- 기획 필요: decisions.md 설정 화면 + technical-design.md Template 모델에 order 필드 + 순서 저장 API
- 기술 참고: 프론트에 @dnd-kit/core 등 드래그 라이브러리 필요 (새 의존성)

- 기술 참고: Slack chat.update API 사용 (봇 메시지는 시간 제한 없이 수정 가능)
